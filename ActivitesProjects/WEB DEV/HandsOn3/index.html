Welcome to the Simple Style Horizon

This standards compliant, simple, fixed width template is released as an 'open source' design (under a Creative Commons Attribution 2.0 Licence), which means that you are free to download and use it for anything you want (including modifying and amending it). All I ask is that you leave the design from HTML5webtemplates.co.uk link in the footer of the template, but other than that...

This template is written entirely in HTML5 and CSS, and can be validated using the links in the footer.
